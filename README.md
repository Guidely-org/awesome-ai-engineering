# Awesome AI Engineering

A curated list of truly awesome AI engineering learning resources, frameworks and libraries.

## Learning resources

### Introduction to AI

Understand the big picture and the key concepts that define the field.

- [Introduction to AI – University of Helsinki](https://course.elementsofai.com/)

### Python programming

Your core tool for building AI systems.

- [Python for Everybody - Chuck Severance](https://www.youtube.com/playlist?list=PLlRFEj9H3Oj7Bp8-DfGpfAfDBiblRfl5p)
- [Automate the Boring Stuff with Python](https://automatetheboringstuff.com/)

**Paid alternatives**

- [Introduction to Computer Science and Programming - edX MITx](https://www.edx.org/learn/computer-science/massachusetts-institute-of-technology-introduction-to-computer-science-and-programming-using-python)
- [Introduction to Computational Thinking and Data Science - edX MITx](https://www.edx.org/learn/computer-science/massachusetts-institute-of-technology-introduction-to-computational-thinking-and-data-science)
